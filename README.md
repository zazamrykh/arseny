## Что такое сеть Хопфилда?

**Сеть Хопфилда** — это рекуррентная нейронная сеть с симметричными весами, работающая как **ассоциативная память**. Она запоминает набор паттернов и умеет восстанавливать их из зашумлённых или неполных входов.

### Ключевые свойства

| Свойство | Описание |
|---|---|
| Число нейронов | N = 784 (28×28 пикселей) |
| Состояния нейронов | Биполярные: **+1** (белый) / **−1** (чёрный) |
| Матрица весов | W (784×784), симметричная, нулевая диагональ |
| Обучение | Правило Хебба — **однократный** проход по паттернам |
| Вспоминание | Асинхронное обновление нейронов до сходимости |
| Передаточная функция | `sign(uᵢ − θ)`, где θ — настраиваемый порог |

### Правило Хебба (обучение)

```
W = (1/N) · Σₚ ξₚ ξₚᵀ,   диагональ W = 0
```

Каждый запомненный паттерн `ξₚ` (биполярный вектор) вносит вклад в матрицу весов.

### Асинхронное обновление (вспоминание)

```
sᵢ(t+1) = sign( Σⱼ Wᵢⱼ sⱼ(t) − θ )
```

Нейроны обновляются в случайном порядке до тех пор, пока состояние сети не стабилизируется (аттрактор).

### Энергетическая функция

```
E = −½ sᵀ W s
```

Сеть всегда движется в сторону уменьшения энергии. Запомненные паттерны — это **локальные минимумы** энергии.

### Теоретическая ёмкость

Сеть из N нейронов надёжно хранит до **≈ 0.138 · N** паттернов.
Для N = 784: **≈ 108 паттернов**.

## Классификация

1. Для каждого класса (буквы) вычисляется **прототип** — усреднённый биполярный вектор всех обучающих примеров класса.
2. Прототипы запоминаются сетью (правило Хебба).
3. При классификации зашумлённый вход подаётся в сеть → recall → сравнение с прототипами по **расстоянию Хэмминга** → предсказанный класс.

## Изменяемые параметры (эксперименты)

### Эксперимент 1 — Порог θ передаточной функции

Варьируется порог θ ∈ {−1.0, −0.5, −0.2, 0.0, +0.2, +0.5, +1.0}.

- θ = 0 — стандартная функция знака
- θ > 0 — нейрон «труднее» переключается в +1 (смещение к −1)
- θ < 0 — нейрон «легче» переключается в +1 (смещение к +1)

**Результат:** `exp1_threshold.png`

### Эксперимент 2 — Ns (число запоминаемых паттернов / ёмкость)

Варьируется число классов Ns от 2 до K (все 26 букв).

- При малом Ns сеть работает надёжно (мало паттернов → мало интерференции)
- При Ns > 0.138·N начинается деградация памяти (ложные аттракторы)

**Результат:** `exp2_capacity.png`
